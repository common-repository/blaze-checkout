== Description ==
 
Install the Checkout button that increases conversion, boosts sales and delights customers.

Blaze Checkout enables a hassle-free, secure one-click checkout experience for your customers. Install it in minutes so your customers can check out in seconds.

Simply install & configure the Blaze Checkout for WooCommerce WordPress plugin to start seeing fewer abandoned carts and more sales for your business. Our top sellers have seen a 20-35% uplift in conversion within one week of installing Blaze Checkout.

== Installation ==

### INSTALL BLAZE CHECKOUT FROM WITHIN WORDPRESS

1. Visit the plugins page within your dashboard and select Add New;
1. Search for Blaze Checkout;
1. Activate Blaze Checkout from your Plugins page;
1. Go to after activation below.

### INSTALL Blaze Checkout MANUALLY

1. Upload the blaze-checkout folder to the /wp-content/plugins/ directory;
1. Activate the Blaze Checkout plugin through the Plugins menu in WordPress;
1. Go to after activation below.

### AFTER ACTIVATION

1. You should see (a notice to start) the Blaze Checkout configuration workout;
